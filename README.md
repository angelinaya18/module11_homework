# module11_homework
Homework module 11
Настроить тестовое окружение JEST и протестировать любую реализованную функцию из тренажера JavaScript, раздел «Функции»:
1) добавить ее в репозиторий;
2) продумать тест-кейсы (успешное/неуспешное выполнение функции);
3) продумать корнер-кейсы; 
4) написать unit-тесты (минимум 3).

Реализованная функция (задание 4.5 из тренажера JavaScript):
Напишите функцию getMonth(n), которая принимает номер месяца и возвращает его название. Например, getMonth(12) ⇒ 'декабрь'. Используйте if/else.
